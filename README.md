# HANKPolicies

This repository contains a small Julia project for generating and reporting
policy-function and value-function figures for a heterogeneous-agent New
Keynesian (HANK) style model.

The project does not contain a separate raw-data directory. The model objects,
policy functions, and plots are generated by the Julia code in `src/` and
`scripts/`.

## Main Files

- `src/HANKPolicies.jl` contains the project module and core model routines.
- `scripts/hank-figures.jl` generates the final report figures saved in
  `figures/`.
- `scripts/policy_value_plots.jl` generates additional diagnostic plots saved in
  `output/`.
- `report.qmd` is the Quarto report. It uses the figures already stored in
  `figures/` and references the `output/` plots as supplementary diagnostics.

## Figures

The figures used in the main report are:

- `figures/figure1.png` and `figures/figure1.pdf`
- `figures/figure2.png` and `figures/figure2.pdf`

The `output/` directory stores generated policy and value plots:

- consumption policy by assets and by cash-on-hand
- labor policy by assets and by cash-on-hand
- savings policy by assets
- flow utility by assets
- value function by assets

These files are useful for checking the model solution and understanding the
components behind the final report figures.

## Reproducing the Project

From the repository root, instantiate the Julia environment:

```bash
julia --project=. -e 'using Pkg; Pkg.instantiate()'
```

Generate the final report figures:

```bash
julia --project=. scripts/hank-figures.jl
```

Generate the supplementary policy and value plots:

```bash
julia --project=. scripts/policy_value_plots.jl
```

Run the tests:

```bash
julia --project=. -e 'using Pkg; Pkg.test()'
```

Render the report with Quarto:

```bash
quarto render report.qmd
```

## Notes

- Report-ready figures are stored in `figures/`.
- Supplementary generated plots are stored in `output/`.
- No external data files are currently present in the repository.
